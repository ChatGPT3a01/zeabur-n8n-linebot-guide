### 📋 匯入步驟

1. **下載 JSON 檔案**
   - 點擊上方下載按鈕
   - 儲存 `.json` 檔案到電腦

2. **匯入到 n8n**
   - 登入你的 n8n 管理介面
   - 點擊右上角 `⋮` → `Import from File`
   - 選擇下載的 JSON 檔案
   - 點擊 `Import` 完成匯入

3. **設定憑證**
   - 根據工作流需求設定相應的憑證
   - 例如：LINE Messaging API、天氣 API 等

4. **測試執行**
   - 點擊 `Execute Workflow` 測試
   - 確認運作正常後啟用（Active）

<br>

> 💡 **提示**：這些模板是課程的延伸實作範例，建議完成 Part 1-4 後再使用！
