# 📥 n8n 工作流模板

這個資料夾包含 3 個實用的 n8n 工作流模板，供學生下載使用。

## 📋 模板列表

### 1. 🌤️ LINE 天氣查詢機器人
**檔案**：`n8n_問Line天氣模板.json`

**功能**：
- 透過 LINE 聊天機器人查詢台灣各縣市即時天氣
- 自動辨識使用者輸入的城市名稱（支援繁體中文）
- 串接天氣 API 取得即時氣象資料
- 自動回覆天氣資訊到 LINE

**使用前需準備**：
- LINE Messaging API 憑證
- 天氣 API Key（如 OpenWeatherMap）

---

### 2. 📰 RSS 爬蟲工作流
**檔案**：`n8n_RSS爬蟲工作流.json`

**功能**：
- 自動訂閱並抓取 RSS 摘要
- 過濾和處理新文章資料
- 定時執行，自動檢查更新
- 可整合推送到 LINE、Slack 等平台

**使用前需準備**：
- RSS 摘要網址
- （選用）推播平台憑證

---

### 3. 📊 每日摘要工作流
**檔案**：`n8n_完整每日摘要工作流.json`

**功能**：
- 整合多個資料來源（新聞、天氣、行事曆等）
- 自動產生每日資訊摘要
- 定時推播到指定平台
- 可客製化摘要內容格式

**使用前需準備**：
- 各資料來源的 API 憑證
- 推播平台憑證（LINE、Email 等）

---

## 💡 匯入方法

1. **下載模板檔案**
   - 從 GitHub 下載 `.json` 檔案

2. **開啟 n8n**
   - 登入你的 n8n 管理介面（在 Zeabur 部署的網址）

3. **匯入工作流**
   - 點擊右上角「⋮」（三個點）
   - 選擇「Import from File」
   - 選擇下載的 JSON 檔案
   - 點擊「Import」

4. **設定憑證**
   - 點擊有紅色警告的節點
   - 設定相應的 API 憑證
   - 儲存設定

5. **測試執行**
   - 點擊「Execute Workflow」按鈕
   - 檢查執行結果
   - 確認無誤後，啟用工作流（開啟 Active 開關）

---

## ⚠️ 注意事項

- 這些模板需要先完成課程 Part 1-3 的設定
- 匯入後請務必檢查並更新所有憑證
- 某些節點可能需要安裝額外的 n8n 社群套件
- 建議先在測試環境執行，確認無誤後再正式使用
